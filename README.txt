
FBX SDK: 2016.1.1 vs2015
RUN IN x86

RENDER CONTROL:

5 for blend animation
6 for toggle render mesh
7 for toggle render joint
(default all true)

ANIMATION CONTROL:

8 for frame based animation
9 for switch model
0 for toggle animation loop mode

CAMERA:

(W,S) for Z axis
(A,D) for X axis
(SpaceBar, Ctrl) for Y axis
Hold Shift for faster movement

LIGHTS:

	DIRECTIONAL LIGHT
	
	hold 1 at all times to control the light
	(I,K) for Z axis
	(J,L) for X axis
	(U, O) for Y axis

	POINT LIGHT
	
	hold 2 at all times to control the light
	(I,K) for Z axis
	(J,L) for X axis
	(U, O) for Y axis

	SPOT LIGHT
	
	hold 3 at all times to control the light
	(I,K) for Z axis
	(J,L) for X axis
	(U, O) for Y axis

CHANGE SKYBOX:
P for switch skybox
